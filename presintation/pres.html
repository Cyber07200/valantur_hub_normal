Идеи для развития

  • Система достижений и бейджей для волонтеров
  • Чат между участниками мероприятия
  • Интеграция с картами для построения маршрутов
  • Push-уведомления о новых мероприятиях поблизости
  • Геймификация: еженедельные челленджи и награды
  • Веб-версия с PWA для десктопа
